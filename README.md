# Quick Start

## Dependencies

- [SDL2 2.0.9+](https://www.libsdl.org/)
- [FreeType 2.13.0+](https://freetype.org/)
- [GLEW 2.1.0+](https://glew.sourceforge.net/)

## POSIX

```console
$ ./build.sh
$ ./ded src\main.c
```

## Windows MSVC

```console
> .\setup_dependencies.bat
> .\build_msvc.bat
> .\ded.exe src\main.c
```
